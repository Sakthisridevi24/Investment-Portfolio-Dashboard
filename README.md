# 📊 Investment Portfolio Dashboard

## 📌 Project Overview
This Power BI dashboard provides a comprehensive analysis of an investment portfolio by tracking portfolio value, returns, dividend yield, asset allocation, and risk distribution. It enables users to make data-driven investment decisions through interactive visualizations.

---

## 🎯 Objectives
- Monitor Total Portfolio Value
- Analyze Overall Return %
- Track Average Dividend Yield
- Compare Asset Class Performance
- Analyze Portfolio Risk Distribution
- Monitor Investment Trends Over Time

---

## 📈 Key Performance Indicators (KPIs)
- 💰 Total Portfolio Value
- 📊 Overall Return %
- 💵 Total Gain/Loss
- 📈 Average Dividend Yield

---

## 📊 Dashboard Features

- Executive Overview
- Portfolio Performance Analysis
- Risk Analysis
- Asset Allocation Analysis
- Interactive Slicers
  - Year
  - Asset Class
  - Risk Rating

> Changing the **Year**, **Asset Class**, or **Risk Rating** dynamically updates the entire dashboard.

---
